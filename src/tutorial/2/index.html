Make a Basic Webpage


The most basic webpage simply consists of an ‘index.html’ file containing some HTML code. Let’s start off by creating a project directory that will contain our code as well as any config files. Let's also make a ‘public_html’ folder to contain our ‘index.html’ file as that is where we will tell Apache/Docker to look for it.

There are plenty of tutorials online that cover everything from basic to advanced HTML. For now, a basic “Hello world!” placeholder page will do.

index.html 
<!DOCTYPE html>
<html>
    <head>
        <title>Example</title>
    </head>
    <body>
        <p>Hello world!</p>
    </body>
</html>

While we can make this file using a graphical text editor, it is useful to know how to use a command-line text editor as it is common to work with GUI-less Linux servers in the development/IT world and in such cases, tools like Vi, Vim, or Nano are the only option. As Vi is universally the most common built-in command-line text editor, we have dedicated a page to using it in the ‘Tools’ section. Learn more about Vi
