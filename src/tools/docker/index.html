Overview

Docker is a set of platform as a service products that use OS-level virtualization to deliver software in packages called containers. Containers are isolated from one another and bundle their own software, libraries and configuration files; they can communicate with each other through well-defined channels.

Installation

While a full guide can be found here - https://docs.docker.com/desktop/windows/install/ , here's a couple of steps to intall it on your own.
1. The First step is to download Docker for your operating system of choice
2. Once thats done, Start up the app
3. Continue with the quick start guide
4. Finally create an account and you're all good to go

Commands

docker stop CONTAINERNAME = stops the webserver
docker rm CONTAINERNAME = removes the container
docker prune = releases the resources that were used for container groups (solves the issue with reaching the quota of container groups)
docker build -t CONTAINER-NAME . = creates a container with the name CONTAINER-NAME
docker run -dp x:x CONTAINER-NAME = runs the container name CONTAINER-NAME on the port x (ie 80:80 runs on port 80)
docker ps = displays information about the containers running such as their ID
docker push docker/CONTAINER-NAME = pushes the container with the tag CONTAINER-NAME to your docker repository
docker login -u YOUR-USER-NAME = logs in to the docke account that you put in “YOUR-USER-NAME”
docker tag CONTAINER-NAME YOUR-USER-NAME/CONTAINER-NAME = gives your container the tag CONTAINER-NAME so that it can be seen and called.
